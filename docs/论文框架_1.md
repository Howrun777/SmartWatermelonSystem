# 毕业设计论文大纲

**论文题目：基于物联网与多光谱技术的智能西瓜无损糖度检测系统设计与实现**

## 摘要

- **研究背景与意义**：传统西瓜糖度检测具有破坏性、抽样率低等缺点，智慧农业需要无损、高效的在线监测方案。
- **核心技术与方法**：采用 ESP32 边缘计算网关结合 AS7341 多光谱传感器，利用多元线性回归（MLR）算法建立糖度预测模型。构建“端-管-云”三层物联网架构。
- **主要创新与成果**：
  - 设计了具备“离线断点续传”与“防雪崩错峰补发”机制的高可用边缘节点。
  - 实现了基于硬件 RTC 与网络 NTP 双重冗余的绝对时间同步中枢。
  - 开发了基于 C++ (cpp-httplib) 的高性能安全后端与 Vue/原生 JS 融合的现代可视化大屏。
- **结论**：系统运行稳定，测糖精度满足实际生产需求，具备良好的商业落地与推广价值。

**关键词**：无损检测；多光谱传感器；物联网；边缘计算；ESP32；数据可视化
