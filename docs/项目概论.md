# 结束后智能西瓜无损糖度检测系统（SmartWatermelonSystem）

## 一、项目概述

智能西瓜无损糖度检测系统是一款面向瓜田生产场景的物联网监测解决方案。系统通过 ESP32 边缘采集终端对西瓜进行多光谱扫描，结合服务端 MLR（多元线性回归）算法实时计算糖度与成熟度，并经由 Web 可视化平台向管理人员展示瓜田整体状况，实现西瓜品质的全程无损监测。

---

## 二、系统架构

系统采用**三层架构**设计，感知层、服务层、应用层职责完全解耦：


| 层级  | 名称                                  | 职责                                | 技术栈                            |
| --- | ----------------------------------- | --------------------------------- | ------------------------------ |
| 感知层 | 瓜田环境采集终端（FieldAcquisitionTerminal）  | 光谱/温湿度/光照数据采集，本地计算糖度，TFT显示，HTTP上传 | ESP32、LVGL、AS7341、SHT31、BH1750 |
| 服务层 | 瓜田数据处理中心（FieldDataProcessingServer） | 设备认证，数据校验，MLR糖度计算，MySQL存储，接口提供    | C++、httplib、libsodium、MySQL    |
| 应用层 | 瓜田监测可视化平台（FieldMonitoringPlatform）  | 前台展示，管理员认证，数据可视化，实时刷新             | HTML/CSS/JS、ECharts            |

## 三、核心功能模块

### 3.1 感知层——瓜田环境采集终端

**传感器数据采集**

- 多光谱采集：AS7341 传感器采集 10 通道光谱数据（415nm~680nm + Clear + NIR）
- 温湿度采集：SHT31 高精度温湿度传感器（精度 ±0.2℃）
- 光照采集：BH1750 光照强度传感器（量程 0~65535 Lux）

**本地计算与显示**

- 本地 MLR 离线糖度计算，适用无网络场景
- 双模式系统：
  - **测试模式（消费者模式）**：单次检测，显示糖度、成熟度、历史均值
  - **工业模式（生产采集模式）**：自动定时检测（5 分钟间隔），本地存储，完整上传

**网络通信与存储**

- WiFi 自动连接与断线重连（每 30 秒）
- HTTP 数据上传（设备编号 + 光谱 + 温湿度 + 光照）
- 时间同步握手（首次联网获取服务器时间, 每次数据传输结束后都会同步时间）
- 离线数据队列：本地持久化存储，网络恢复后补发（随机延迟 0~10 秒防雪崩）

**硬件配置**

- ESP32 DevKit V1 主控（双核 240MHz，4MB Flash）
- 2.8 寸 TFT 显示屏（ILI9341，SPI 接口）+ XPT2046 触摸
- DS3231 RTC 时钟模块（可选）

### 3.2 服务层——瓜田数据处理中心

**数据接收与校验**

- 设备认证：Header 中 device_id + token 校验，`device_auth` 表匹配且 status=1
- 环境数据校验：温度/湿度/光照不能全为 99，数值范围合法性检查
- 瓜田号合法性校验：匹配 `field_production` 表

**核心算法**

- 糖度计算（MLR 多元线性回归）：
`Sugar = 0.0012 × ch415 + 0.0005 × ch445 + 0.0021 × ch480 + 5.2`
- 成熟度计算：
`maturity_score = sugar_brix / mature_sugar_threshold`

**数据库存储**

- 5 张核心数据表：`admin_users`、`device_auth`、`watermelon_data`、`field_production`、`field_environment`

**安全认证**

- 管理员密码哈希：libsodium Argon2id，防暴力破解
- 会话管理：UUID Session（32 位十六进制），1 小时 TTL
- 设备认证：Token + 状态码双重校验

**接口服务**

- 设备端接口：数据上传 `/api/device/upload`、时间同步 `/api/device/time`
- 管理端接口：登录 `/api/admin/login`、瓜田列表、西瓜数据查询、环境数据查询、历史数据查询

### 3.3 应用层——瓜田监测可视化平台

**监控大屏**

- 左侧：西瓜数据阵列（编号/成熟度/糖度），掉线告警（30 分钟无数据标记红色）
- 左侧：单西瓜糖度历史折线图（ECharts）
- 右侧：瓜田环境折线图（温度/湿度/光照，ECharts）
- 瓜田分页导航 + 60 秒无操作自动切换瓜田巡航
- 每 6 秒静默刷新图表数据

**登录认证**

- Session + Cookie 会话管理
- 角色区分：超级管理员 / 普通管理员

## 五、数据库设计


| 表名                  | 用途                             |
| ------------------- | ------------------------------ |
| `admin_users`       | 管理员账户信息（Argon2 哈希密码）           |
| `device_auth`       | 设备身份认证（device_id、token、status） |
| `watermelon_data`   | 西瓜光谱/糖度/成熟度数据                  |
| `field_production`  | 瓜田生产信息（品种、成熟糖度阈值）              |
| `field_environment` | 瓜田环境监测数据（温度/湿度/光照）             |

## 六、关键业务规则

- **环境数据无效**：温度=99 AND 湿度=99 AND 光照=99 → 环境数据跳过写入，西瓜数据不受影响
- **糖度算法**：`Sugar = 0.0012×ch415 + 0.0005×ch445 + 0.0021×ch480 + 5.2`
- **成熟度**：`maturity_score = sugar_brix / mature_sugar_threshold`
- **设备禁用**：`status=0` → HTTP 401 拒绝上传
- **离线补发**：网络恢复后历史数据补发，随机延迟 0~10 秒防止雪崩
